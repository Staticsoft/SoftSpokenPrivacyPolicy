# Privacy Policy

## 1. Introduction
This policy describes the data processed, the purposes of processing, retention rules, data sharing, and user rights.

## 2. Data Collected
**Account data:** Email address and password stored in AWS Cognito.  
**Subscription data:** Stripe customer identifier and subscription status.  
**Operational data:** Etsy message content submitted through the extension.  
**Store inputs:** Custom information provided by the user to improve response generation.

## 3. Purpose of Processing
Account data is used for authentication and access control.  
Subscription data is used to verify entitlement to usage and to initiate checkout sessions.  
Etsy messages and store inputs are used exclusively to generate responses via OpenAI models.

## 4. Retention
Account data in AWS Cognito is retained indefinitely until the user requests deletion.  
Subscription data is retained according to Stripe’s requirements.  
Etsy messages and store inputs are retained only for the duration of a generation request and deleted immediately afterward.

## 5. Data Sharing
Etsy messages and store inputs are shared with OpenAI to generate responses.  
Account data and subscription data are not shared with OpenAI.  
No data is sold or shared with other third parties.

## 6. Storage and Security
Account data is stored in AWS Cognito.  
All data transfers occur over encrypted channels.  
Access is limited to service components and personnel requiring it for operation.

## 7. User Rights
Users may request access to or deletion of AWS Cognito account data.  
Users may request deletion of Stripe-related records.  
Etsy messages and store inputs cannot be accessed after processing due to immediate deletion.

## 8. International Transfers
Data may be processed or stored in regions used by AWS Cognito, Stripe, and OpenAI.

## 9. Children
The service is not intended for individuals below the age required by applicable law and does not knowingly process children’s data.
